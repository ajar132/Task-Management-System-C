# Task-Management-System-C
🚀 Advanced Task Management System (C Language)
A robust command-line application designed to manage tasks efficiently using advanced data structures. This project was developed as part of the Advanced Programming and Data Structures module at EMSI.

📑 Project Overview 
The main goal of this project is to provide a flexible system to organize, prioritize, and track tasks while maintaining high performance through dynamic memory management.

✨ Key FeaturesDynamic 
Task Management: Add, modify, search, and delete tasks with ease.Undo Functionality (Stack-based): Accidentally deleted a task? The system uses a Stack to store deleted items, allowing you to undo your last deletion.Priority Queue Sorting: Tasks can be automatically sorted and displayed based on their urgency levels.Double Linked List Implementation: Ensures efficient navigation and manipulation of the task list.

🛠️ Technical Specifications
Language: C.
Data Structures: Double Linked Lists: For the core task storage.Stacks: For the "Undo" feature.Priority Queues: For task sorting.
Concepts: Pointer manipulation, Dynamic Memory Allocation (malloc, free), and Algorithm optimization.

📐 Software Design (UML)
This project follows a professional engineering workflow. Before coding, the system was modeled using UML diagrams to ensure a clean architecture:
Activity Diagram: To model the business logic and user flow.
Class Diagram: To define the relationships between Tasks, Nodes, Stacks, and Queues.
Object Diagram: To show a snapshot view of the running system, with concrete objects.

🚀 How to Run
Clone the repository: git clone https://github.com/ajar132/Task-Management-System-C.git
Compile the code: cc main.c -o task_manager
Run the application: ./task_manager
